## Image Features

### Query Parameters Used:
- `w=XXX` - Width in pixels
- `h=XXX` - Height in pixels
- `fit=crop` - Ensures images are cropped to exact dimensions

### Benefits:
1. **Free to Use**: Unsplash provides free high-quality images
2. **Commercial Use**: Licensed for commercial applications
3. **CDN Delivery**: Fast loading times via Unsplash CDN
4. **Optimized**: Images are automatically optimized
5. **Responsive**: Can adjust size via query parameters

## Replacing Images

### To use your own images:
1. Upload images to `src/assets/images/services/`
2. Update the `servicePlaceholders` object in `home.component.ts`
3. Change URLs from Unsplash to local paths:
   ```typescript
   salon: [
     'assets/images/services/salon-1.jpg',
     'assets/images/services/salon-2.jpg',
     // ... more images
   ]
   ```

### To use different Unsplash images:
1. Visit https://unsplash.com/
2. Search for your desired image
3. Click on the image
4. Click "Download" dropdown
5. Select "Copy link"
6. Paste the link and add query parameters: `?w=200&h=140&fit=crop`

## Image Optimization Tips

1. **Use appropriate dimensions**: Don't use 4K images for thumbnails
2. **Add query parameters**: Unsplash automatically optimizes based on parameters
3. **Lazy loading**: Already implemented with `loading="lazy"` attribute
4. **Error handling**: Fallback images ensure UI never breaks

## Performance

- Images are loaded via CDN (fast delivery)
- Lazy loading prevents blocking page load
- Optimized dimensions reduce bandwidth
- Error handling prevents broken images
